## Description
Nestjs backend with auth and stuff

## Installation

```
$ yarn install
```

## Running the app

    $ docker-compose up

#### Api docs:
  
  [localhost:5000/docs](http://localhost:5000/docs)

#### Api endpoints: 

  [localhost:5000/api](http://localhost:5000/api)

#### Adminer:

  [localhost:8010](http://localhost:8010/)

## Tests

```
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
